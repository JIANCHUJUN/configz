vim有四种模式
正常模式：在文件中四处移动光标进行修改
插入模式：插入文本
替换模式：替换文本
可视化（一般，行，块）模式：选中文本块
命令模式：用于执行命令

按ESC进入正常模式
在正常模式：i = 插入模式，大写R = Replace模式，v = 可视（一般）模式，大写V = 可视（行）模式，ctrl+v = 可视（块）模式，: = 命令模式

命令
:e {文件名} = 打开要编辑的文件（和在terminal中打开文件差不多）
:ls = 列出所有缓存
:b {数字} = 跳转回打开的，对应这个数字的file

正常模式
w = 下一个单词
b = 词初 = back = begin => backward => 光标落在词的首个字符上
e = 词尾 = end => forward => 光标落在词的最后一个字符上
（其实b和e两个结合起来用就够了，w有点多余）


0 = 行初
$ = 行尾
^ = 本行第一个非空字符

H = 屏幕首行 = high
M = 屏幕中间 = middle
L = 屏幕底部 = low

ctrl+u = 上翻页 = up
ctrl+d = 下翻页 = down

gg = 文件开头
G = 文件结尾

:{数字} = 跳转到相应行数

% = 找到匹配

/{正则表达式} = 搜索 = n/N用于导航匹配
O = 在当前光标所在行之上插入一行，并将光标移至上一行的行首，并进入insert模式
o = 在当前光标所在行之下插入一行，并将光标移至下一行的行首，并进入insert模式

u = 撤销
ctrl + r = 反撤销

d{移动命令} = 根据移动命令进行删除
dd = 删除整行

x = 删除目前光标下的字符
i = 替换当前光标下的字符 = 删除字符 + 进入插入模式

y = 复制选中内容
p = 粘贴选中内容

ci( = 删除当前()括号内内容 => i = inside
ci[ = 删除当前[]括号内内容
da' = 删除一个单引号字符串，包括周围的单引号，双引号同理 => a = around

:%s/foo/bar/g = 全局替换
:s/foo/bar/g = 当行替换

newline char in vim = /r 而不是 /n

:sp = 横切分屏
:vsp = 竖切分屏
:q = 退出目前选中的分屏

30j = 往下跳30行
30k = 往上跳30行

f{字符} = 查找并跳转到本行向前出现的这个字符
t{字符} = 查找并跳转到本行向后出现的这个字符

S = 删除本行，把光标移至下一行行首，并进入insert模式

设置vimrc
set tabstop=4

尽量不要复制粘贴别人的整个设置文件， 而是阅读和理解它， 然后采用对你有用的部分。
